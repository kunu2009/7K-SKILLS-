# 7K Skills

**A minimalistic habit stacking app for overstimulated minds.**

Part of the **7K Ecosystem**.

## Features

- **Habit Stacking**: Organize your day into Morning, Afternoon, and Evening stacks.
- **Minimalist Design**: Clean, distraction-free interface with customizable themes.
- **Gamification**: Earn points, streaks, and badges to stay motivated.
- **Progress Tracking**: Visualize your consistency with weekly trends and monthly calendars.
- **Mindfulness**: Built-in sleep countdown and mindful moments.
- **PWA Support**: Installable on mobile and desktop for offline access.

## Tech Stack

- **React** (Vite)
- **TypeScript**
- **Tailwind CSS**
- **Framer Motion**
- **Recharts**
- **Lucide React**

## Getting Started

1.  Clone the repository.
2.  Install dependencies: `npm install`
3.  Run the development server: `npm run dev`
